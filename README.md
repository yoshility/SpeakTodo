# SpeakTodo

![feature画像](./assets/playstore_feature.png)

- 音声でTodoを入力できて、かつ通知欄にTodoを残すことで忘却防止してくれるシンプルアプリ
- 音声入力なので、外出時にふとタスクを思い出したときに手軽にメモできる

## 特徴

- 音声Todo入力（日本語）
- キーボードTodo入力
- 通知欄にTodoを残す
- クラウドを使っていないのでログイン不要
- データはスマホのローカルに保存
- Androidにのみ対応
